# セルフレジシステムの再現
ITエンジニアを目指し、現在プログラミングを学習しています。  
Pythonでの学習状況を確認することを目的にセルフレジのシステムを再現しました。  

# 使い方
ウィンドウ下部に「商品バーコード(JANコード付き)画像」をドロップ、またはJANコードを入力してボタンをクリックしてください。  
データの照合に成功すると商品データがウィンドウ中部表示され、ウィンドウ上部の履歴が更新されます。  
データは「株式会社タジマヤ( https://www.tajimaya-cc.net/products )」 とスクレイピングすることで照会しています。
